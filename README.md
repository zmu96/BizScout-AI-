# 📊 BizScout  
AI 기반 기업 분석 웹 서비스 (Prototype)

---

## 1. 프로젝트 개요
BizScout는 기업명을 입력하면 해당 기업의 기본 정보, 최근 이슈, 채용 정보,
기업 분위기 등을 AI 분석 결과 기반으로 요약하여 제공하는 웹 서비스 프로토타입입니다.

본 프로젝트는 AI를 활용한 정보 요약 서비스의 구조와 데이터 흐름을 검증하는 것을
목표로 하며, 프론트엔드에서 AI 분석 결과를 시각화하는 과정에 중점을 두었습니다.

---

## 2. 기획 배경 및 문제 정의
취업 준비 과정에서 기업 정보를 조사할 때,
기업 개요, 최근 뉴스, 채용 정보, 기업 문화 등이 여러 사이트에 분산되어 있어
정보를 종합하는 데 많은 시간이 소요됩니다.

본 프로젝트는 이러한 문제를 해결하기 위해
AI를 활용하여 기업 관련 정보를 요약하고,
한 화면에서 확인할 수 있는 서비스를 기획하였습니다.

---

## 3. 주요 기능
- 기업 기본 정보 요약 (기업명, 산업, 본사, 설립 연도)
- 긍·부정 키워드 기반 분석
- 최근 기업 뉴스 요약 및 링크 제공
- 기업 관련 짧은 인사이트 제공
- 채용 정보 및 관련 링크 제공

---

## 4. 시스템 구성 및 아키텍처
본 서비스는 AI 기반 기업 분석 웹 서비스의 구조를 검증하기 위한 프로토타입으로,
다음과 같은 구성 요소로 설계되었습니다.

- **프론트엔드**
  - React, Bootstrap
  - 사용자로부터 기업명을 입력받고,
    AI 분석 결과(JSON)를 대시보드 형태로 시각화

- **백엔드 워크플로우**
  - Docker 환경에서 실행되는 n8n을 활용하여 워크플로우 구성
  - Webhook 기반으로 데이터 흐름을 제어하는 구조로 설계 (Webhook ~ respond to Webhook)

- **AI 분석**
  - LM Studio 환경에서 로컬 LLM 모델을 활용
  - 사용 모델: **kullm-r-i1 (약 4.5GB)**
      kullm-r-i1은 한국어 데이터를 기반으로 학습된 모델로,
      기업명, 산업 용어, 취업·채용 관련 표현 등
      한국어 맥락을 이해하는 데 강점이 있습니다.
  - 기업 관련 비정형 데이터(텍스트, HTML 등)를 분석하여
    웹 서비스에서 바로 활용 가능한 JSON 형태로 구조화

- **데이터 교환 형식**
  - 프론트엔드와 백엔드 간 데이터는
    사전에 정의된 JSON 스키마를 기준으로 교환

전체 시스템은
`사용자 입력 → (워크플로우 처리 → AI 분석 → JSON 결과 생성) → 프론트엔드 렌더링`
의 흐름으로 동작하도록 설계되었습니다.

  },
  "newsSummary": "",
      "link": "",
  "insight": [],
  "jobs": [
    {
      "title": "",

  ]
